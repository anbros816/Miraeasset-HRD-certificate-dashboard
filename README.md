# 자격증 취득지원 현황 대시보드 | 미래에셋증권 HRD팀

임직원 자격증 취득지원 신청·승인 현황 대시보드.
**데이터는 GitHub에 올리지 않고, 대시보드 화면에서 누적 엑셀을 직접 업로드**해서 사용합니다.

## 사용 방법 (매월)

1. 배포된 대시보드 주소 접속
2. 첫 화면의 업로드 박스에 **누적 엑셀 파일을 드래그** (또는 클릭해서 선택)
3. 끝. 6월 데이터가 추가된 누적 파일을 올리면 6월까지 자동 반영됩니다
4. 사용 중 갱신하려면 상단 **[엑셀 업로드]** 버튼으로 언제든 재업로드

※ 엑셀 파일은 **브라우저 안에서만 처리**되며 서버·GitHub 어디에도 전송/저장되지 않습니다.
따라서 실명·사번이 포함된 원본 파일을 그대로 사용해도 외부 유출 위험이 없습니다.

## 엑셀 양식 (열 순서 고정, 1행은 제목행)

| 열 | 내용 | 열 | 내용 |
|---|---|---|---|
| A | 번호 | K | 응시료 |
| B | 회사 | L | 교육비 |
| C | 자격증코드 | M | 신청금액 |
| D | 자격증명 ("한시적" 포함 시 자동 분류) | N | 승인금액 |
| E | 발급기관 | O | 세금 |
| F | UID | P | **신청일** (YYYY-MM-DD) ← 월별 집계 기준 |
| G | 성명 | Q | 지급일 |
| H | 사번 | R | 직무관련 |
| I | 직급 | S | 상태 (승인/접수/반려/취소) |
| J | 부서 | T | 비고 |

## 저장소 구성

- `index.html` — 대시보드 전체 (이 파일 하나면 동작)
- 대시보드 개선 시 `index.html`만 교체 업로드하면 됩니다
